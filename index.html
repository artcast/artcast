The Artcast Project

This is the Artcast Project: a set of resources for data-driven artists. First, we provide the Artcast Server - a special-purpose web server that is dedicated to the task of delivering time series data for use in:

  • Interactive artwork
  • Kinetic sculpture
  • Maker projects
  • Cool hacks
  • School projects
  • Whatever you dream up!
Second, we provide a publicly accessible instance of the Artcast Server, running at artcastproject.org. Anyone can use the data feeds at artcastproject.org to create their own projects.
